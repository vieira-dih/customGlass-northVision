# 🕶️ NorthVision - Customizador de Óculos (Frontend)

Interface web desenvolvida em **React** para um sistema de personalização de óculos.

O usuário pode escolher um modelo e personalizar as cores das lentes diretamente na interface.

---

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS3
- React Router

---

## 🎯 Funcionalidades

- Listagem de modelos de óculos
- Navegação com rotas dinâmicas
- Página de customização de produto
- Alteração de lentes em tempo real
- Interface responsiva
