# 🕶️ NorthVision - Customizador de Óculos (Frontend)

Interface web desenvolvida em **React** para um sistema de personalização de óculos.

O usuário pode escolher um modelo e personalizar as cores das lentes diretamente na interface.

---

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS3
- React Router

---

## 🎯 Funcionalidades

- Listagem de modelos de óculos
- Navegação com rotas dinâmicas
- Página de customização de produto
- Alteração de lentes em tempo real
- Interface responsiva

## ▶️ Como rodar o projeto

### 1️⃣ Clonar o repositório


``` bash
https://github.com/vieira-dih/customGlass-northVision.git

```


### 2️⃣ Entrar na pasta
``` bash
cd frontend

```

### 3️⃣ Instalar dependências

``` bash
npm install
```

### 4️⃣ Rodar o projeto

``` bash
npm run dev
```


O projeto ficará disponível em:

http://localhost:5173
